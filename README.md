Description:
This project deploys a static website using AWS S3 for storage and CloudFront as a CDN.

Steps performed:
1. Created an S3 bucket.
2. Uploaded website files (HTML, CSS, JS, images).
3. Configured CloudFront distribution.
4. Set index.html as the default root object.
5. Accessed the website using the CloudFront URL.
